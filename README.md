
# react-spa
A React single page application built using real APIs.  Includes modern UI features such as infinite scroll, protected routes, multi-language support, nested comments, accordion, image carousel, offset pagination, YouTube-style live chat with debouncing &amp; caching, and an autocomplete search bar.

• Navbar & routing with dedicated pages
• Infinite scroll on Home page using real APIs
• Protected routes with Login authentication
• Multi-language support using i18n on About page
• Accordion component
• Nested comments UI & recursion
• Image slider / carousel component
• Offset pagination with API
• YouTube live chat simulation with auto-scroll
• Autocomplete search bar with debouncing + caching

Includes reusable components, custom hooks, and clean folder structure.
